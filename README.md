## About

**ShokherSrity** *(Bengali: শখের স্মৃতি — "memories of love")* is a premium wedding photography and cinematography studio based in **Bhanga, Faridpur, Bangladesh**, serving couples across **Dhaka, Faridpur,** and the entire country.

We specialize in capturing the timeless beauty of Bangladeshi weddings — every glance, every laugh, every quiet moment between two people in love.

## Run Locally

No build tools required — just serve the folder.

```bash
# Python 3
python3 -m http.server 5000

# or Node.js
npx serve -l 5000

# or any static server you prefer
```

Then open [`http://localhost:5000`](http://localhost:5000) in your browser.

## Search Engine Optimization

Every page ships with a complete SEO head block, designed to rank for **wedding photography in Bangladesh**.

- Keyword-rich `<title>` and `<meta description>` per page
- Brand keywords in English, romanized Bengali, and **শখের স্মৃতি**
- Geo tags pinning the studio to Bhanga, Faridpur, Bangladesh
- **Open Graph** and **Twitter Card** meta for rich link previews on Facebook, WhatsApp, and X
- Canonical URLs to prevent duplicate-content penalties
- **Schema.org JSON-LD** structured data:
  - `LocalBusiness` / `ProfessionalService` on the homepage
  - `ContactPage` and `FAQPage` on the contact page
  - `ImageGallery` on the gallery
  - `Service` with `OfferCatalog` on packages
  - `CollectionPage` on reels
- `robots.txt` + `sitemap.xml` for crawler-friendliness
- Verified with **Google Search Console**
